# Crowdspark

A landing page prototype for the Crowdspark app experience.

## Structure

- `Crowdspark/index.html` — main HTML file for the prototype
- `.gitignore` — ignores common OS, editor, and node artifacts
- `.github/workflows/pages.yml` — GitHub Pages deployment workflow

## Usage

1. Open `Crowdspark/index.html` in your browser.
2. Edit the file directly to update the design and behavior.

## GitHub Pages

This repo uses GitHub Actions to deploy the `Crowdspark` folder as a static site on push to `main`.

After the first successful run, your site will be available at:

`https://DaveTmx.github.io/Crowdspark/`

## Notes

This repository currently contains a single static HTML prototype.

Roles & Permissions

RBAC matrix — capabilities by role

ViewerDefault role — all new registrations
  • Browse all published resources
  • Download files
  • Search and filter
  • Upload new resources
  • Edit or delete resources
  • Access admin panel
UploaderTrusted contributors — approved by admin
  • All Viewer permissions
  • Upload new resources
  • Edit their own resources
  • Delete their own resources
  • Edit other users' resources
  • Access admin panel
AdminFull platform access
  • All Uploader permissions
  • Edit or delete any resource
  • Manage all user accounts
  • Assign and revoke roles
  • Approve pending accounts
  • Read audit log
